About this project

PHOTOin! adalah aplikasi web photobooth digital yang dikembangkan sebagai proyek mata kuliah Pengembangan Aplikasi Web di Universitas Brawijaya. Dibuat untuk memberikan pengalaman photobooth yang gratis, personal, dan dapat dinikmati siapa saja.
